== Description ==

Use SnagFilms Player Embed to embed a film from [SnagFilms.com](http://www.snagfilms.com/ "") into any page/post on your site.

Embed the SnagFilms Player using shortcode '[snagfilm id="FILM-ID-HERE"]'; replacing 'FILM-ID-HERE' with the id of the desired SnagFilm.

= Useful plugins: =
* ["iframe"](http://wordpress.org/extend/plugins/iframe/ "") - Use iframe to embed video from YouTube or Vimeo or to embed Google Map or just to embed content from external page with a shortcode.
* ["Embedded Video"](http://wordpress.org/extend/plugins/embedded-video-with-link/ "") - Easy embedding of videos from lots of portals or local video files of many file formats with optional generation of a corresponding link.

== Other Notes ==

= SnagFilms Player Embed Parameters: =
* **id** - id of the film you wish to display '[snagfilm id="ea4af620-a748-11e0-a92a-0026bb61d036"]'
* **more coming soon** - please send any requests for new parameters to adam @ snagfilms.com

== Installation ==

1. Install and activate the plugin on the Plugins page.
2. Add the shortcode '[snagfilm id="FILM-ID-HERE"]' to page or post content, replacing 'FILM-ID-HERE' with the id of the desired SnagFilm.
